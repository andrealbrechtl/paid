Introduction

Sinewave is a one page Hosting HTML Landing Page Template with very Modern and Clean Design built with HTML5, CSS3 and JS. It is perfect Choice for your Hosting website Landing page.It is equipped with register and login form also. It can be customized easily to suit your wishes.

SASS Compilation

Requirements:-

  1. Node ver 5.1.1
  2. NPM ver 3.10.2

Place the theme folder in htdocs

On terminal get into the theme folder using cd command

example

$cd /c/xampp/htdocs/sinewave-html

After this run command

$npm install

This will install all the packages used for scss compilation.

Now run command

$gulp

This command will compile scss files to css files.

Watch command run

$gulp watch
